# Student Management System

## 🚀 Features
- Login authentication
- Add, Edit, Delete students
- Search functionality
- MySQL integration

## 🛠️ Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS

## 📌 Description
A full-stack web application to manage student records with authentication and search feature.

## ▶️ How to Run
1. Install dependencies:
   pip install flask mysql-connector-python

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000/
